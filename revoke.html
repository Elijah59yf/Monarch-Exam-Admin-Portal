Revocation Process

  • • Results become immediately inaccessible to students
  • • Notification emails sent to affected students (if configured)
  • • Data remains in database for audit purposes
  • • Revocation timestamp and admin details are logged
  • • Warning: Use only for data corrections or emergencies

⚠️ Critical Notice

Revoking results is a significant action that affects all enrolled students. This action should only be performed when there are data integrity issues or critical errors in the published results.
